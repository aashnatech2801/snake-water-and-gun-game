# 🐍 Snake Water Gun Game
A fun, browser/console-based take on the classic Rock-Paper-Scissors game — Indian style! Play *Snake, Water, Gun* against the computer and see who wins.

## 🎮 Rules
- *Snake* drinks *Water* → Snake wins
- *Water* drowns *Gun* → Water wins
- *Gun* shoots *Snake* → Gun wins
- Same choice on both sides → It's a Draw

## 🛠️ Tech used
-Python

## ✅ Features
- Play against the computer
- Score tracking (Wins / Losses / Draws)
- Simple, responsive interface
